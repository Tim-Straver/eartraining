# Waar zet ik audio bestanden?

Plaats je bestanden hier:

- `app/src/main/assets/chords/` (bijv. `A.wav`, `A#.wav`, `Asus2.wav`, `Asus2 2.wav`)
- `app/src/main/assets/notes/` (bijv. `C.wav`, `D#.wav`, `A# 2.wav`)

De app leest deze mappen automatisch.

- In **Chord Progressions** maakt de app een reeks van meerdere chords uit `assets/chords`.
- In **Chord Types** en **Notes** speelt de app één bestand per vraag.

## Extra naamgeving

- Sharps (`#`) zijn volledig ondersteund.
- Een suffix zoals ` 2` (spatie + nummer) mag gebruikt worden voor extra octaven/varianten.
- Voor de antwoordlabels negeert de app die numerieke suffix.

- De app speelt alle progressions uit 4 losse chords, zodat je niet aan de lengte kunt horen welke het is.
